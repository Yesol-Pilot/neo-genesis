# Code of Conduct

## Our standard

Neo Genesis adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1, as the standard for participation in this project. Please read the full text on the Contributor Covenant website.

## Scope

This standard applies to all project surfaces operated by Neo Genesis, including:

- The `Yesol-Pilot/neo-genesis` GitHub repository (issues, pull requests, discussions, code review comments)
- Hugging Face dataset and Space discussions under `neogenesislab`
- Cross-published surfaces (dev.to, Hashnode, GitHub Discussions) where the community engages with our content

## Reporting

To report a concern, contact the project operator at `help@neogenesis.app` with subject prefix `[CONDUCT]`. Reports are reviewed privately. The operator commits to acknowledging reports within 72 hours and providing an outcome summary within 14 days.

## Enforcement

The project operator is responsible for clarifying the standard and responding to reports. Possible outcomes include a private clarification, an edit request, a temporary participation pause, or removal from the project surfaces above. Decisions are documented in a private log and summarised in the public `CHANGELOG.md` only when the outcome affects published content.
